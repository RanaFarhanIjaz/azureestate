# Azure Estate

A real estate Android app for listing and discovering properties.

## Features
- Browse properties
- Search with filters
- Add to favorites
- Chat with owners
- AI assistant for help
- Upload your own properties

## Tech Stack
- Android (Java)
- Supabase (Database & Storage)
- Firebase (Auth & Chat)
- Groq AI (Chat assistant)

## Setup
1. Open in Android Studio
2. Add your `google-services.json`
3. Update Supabase keys in `SupabaseConfig.java`
4. Add Groq API key in `AiChatFragment.java`
5. Build and run

## API Keys Needed
- Supabase URL & Anon Key
- Firebase google-services.json
- Groq API Key (free from console.groq.com)

## Screens
- 🏠 Home - Property listings
- 🔍 Search - Find properties
- ❤️ Favorites - Saved properties
- 🤖 AI Chat - Ask questions
- 👤 Profile - User info
